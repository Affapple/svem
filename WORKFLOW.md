# Workflow Procedures for CVE-2025-24813 PoC

1. **Todo-First**: All tasks must be recorded in `TODO.md` before execution.
2. **Action-Rationale**: Each task in `TODO.md` must clearly state the action to be taken and the reason (Why) it is necessary.
3. **Execution**: Implement only one task at a time.
4. **Post-Action**: After each implementation, update the task status in `TODO.md` and append a brief summary of results/findings.
5. **Documentation**: Keep logs concise and direct.
